== Description ==

"Premium Matrix Rate is a WooCommerce extension that allows admin to define the custom multiple shipping rates according to need."

= Main features In Free Vesion =

* It allows admin to create multiple table rates.
* Create rate with all conditions at a time like country, region, city, zip code and quantity. 
* With this module admin can allow the free shipping to the product
* Admin can set maximum allowed weight and minimum order amount for free shipping
* Admin can set if virtual product will be used or skipped in shipping rate calculation.
* Easy to use and language support
* Easy to edit rates

= Main features In Premium Vesion =

* It allows admin to create multiple table rates.
* Create rate with all conditions at a time like country, region, city, zip code, weight, cart subtotal and quantity 
* With this module admin can allow the free shipping to the product
* Admin can set maximum allowed weight and minimum order amount for free shipping
* Admin can set if virtual product will be used or skipped in shipping rate calculation.
* Easy to use and language support
* Easy to edit rates
* Easy to import rate

 You can get premium version of "Premium-Matrix-Rate" from this link : http://cedcommerce.com/woocommerce-extensions/premium-matrix-rate
